# Fake News Detector (AI Powered)

An AI-based Fake News Detection web app built with **Next.js 14** and **Tailwind CSS**.  
This application analyzes news articles in real time to help identify potential misinformation using AI-powered insights.

## Features

- **AI-powered detection** of fake news content
- **Real-time text and article analysis**
- **Interactive Dashboard** for data visualization
- **Simple and modern UI** using Tailwind CSS
- **Fast performance** powered by Next.js App Router

## Tech Stack

- **Framework:** Next.js 14
- **Language:** JavaScript (React)
- **Styling:** Tailwind CSS
- **Database:** MongoDB (via Mongoose)
- **AI Integration:** OpenAI

## Installation & Setup

Follow these steps to run the project locally:

```bash
# 1 Clone the repository
git clone https://github.com/KavyaChauhan135/fake-news-detector.git

# 2 Install dependencies
npm install

# 3 Run the development server
npm run dev

# 4 Open in browser
http://localhost:3000
```
